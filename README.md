# Chang Guo — Personal Website

Academic personal website for Chang Guo, a PhD Candidate in Computer Science
at Arizona State University.

## Development

```bash
npm ci
npm run dev
```

The local site runs at `http://localhost:3000`.

## Validation

```bash
npm run lint
npm run build
```

## Content

- Biography and public contact links
- News
- Selected research
- Open-source contributions
- Experience
- Education and awards

The visual direction is inspired by
[Qiaolin Yu's academic homepage](https://qiaolin-yu.github.io/). The
implementation and assets in this project were created independently.
